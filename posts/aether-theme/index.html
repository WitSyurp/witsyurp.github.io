高效、干净的aether主题

Posted May 2, 2018

如今的网络是一个令人沮丧的混乱弹出窗口，侵入性横幅和广告阻碍妨碍浏览内容。aether的设计，摆脱了这些干扰，只是突出内容。网页感觉更像是一个本机应用程序，而不是一个网站。互动是直观的，内容是焦点，干扰被省略。

Aether 是一个Hugo主题的博客，为良好的写作和摄影提高了生产力。如果你用Hugo作为你的博客，请试试看。

安装

在 Hugo 项目的根目录中，将 aether repo 克隆到主题目录中。

git clone https://github.com/josephhutch/aether.git themes/aether

使用

网站配置

如果以下参数尚未存在，请将其添加到配置文件congfig.toml中。

baseURL = "https://yourwebsitenamegoeshere.com"
languageCode = "your language code"
title = "your website title"
theme = "aether"
googleAnalytics = "Your google analytics tracking ID"

[params]
brand = "the title displayed in the nav bar - optional"
description = "your website's description"
homeimg = "url to the image used for the home button - optional"
bgimg = "url to the image used for the background - optional"

title参数用于每个页面标题，即搜索引擎在搜索结果中显示的标题。如果希望页面左上角显示的标题与页面标题不同，请使用brand参数。例如，我的站点的title参数是Joe Hutchinson，但是brand参数设置为joehutch。

homeimg和bgimg参数使您能够进一步自定义站点的外观。homeimg参数是用于每页底部home按钮的图像。由于“主页”按钮上使用的文本为白色，因此首选较暗的背景图像。如果未指定homeimg参数，则使用回退映像。类似地，bgimg参数用于每个网页的背景。aether的设计，以一个微妙的平铺图像为背景，看起来最好。如果未指定背景图像，则背景将为纯色。

这是站点级别所需的唯一配置！你现在可以开始为你的网站写内容了。

创建内容

在shell中执行hugo new posts/postnamehere/index.md，创建一篇新的博客文章。在新的标记文件的顶部，是所谓的前端问题。最重要的是页面的元数据，它决定了Hugo和aether如何为您的文章生成HTML。下面你可以找到一个新的帖子的头条应该包含什么，以及每个参数的含义。

---
title: "文章的标题"
date: 生成帖子的日期
description: "帖子说明（显示在帖子卡片中）"
categories: ["在此处添加逗号分隔的类别”，“其他类别"]
featuredImage: "页面特色图片的url"
featuredImageDescription: "功能图像的说明，用作alt文本"
dropCap: 如果第一个字母应该是一个大的装饰大写字母(true, false)
displayInMenu: 如果希望导航菜单中列出文章(true, false)
displayInList:如果希望文章列在主页和分类页面上 (true, false)
draft: 如果希望页面是草稿 (true, false)
---

displayInMenu和displayInList参数用于确定内容的显示位置。文章通常将displayInMenu设置为false，以便文章不是菜单选项，而displayInList设置为true，以便它显示在主页的文章列表和在分页列表中。另一方面，About Me页面将displayInMenu设置为true，displayInList设置为false。这将允许从菜单访问About Me页面，但不显示在主页的帖子列表中。

categories参数用于对类别页中的类似文章进行分组。类别页面可以从菜单中访问，并列出具有相同类别的所有文章。

dropCap参数用于确定文章的第一个字母是否应该是大写字母。落下的大写字母是书或章节开头的大装饰字母。

在每一篇文章中添加一个有趣的描述和一个好的图片，从这个主题中获得最大的价值。

Aether利用[页面捆绑包]（https://gohugo.io/content-management/page-bundles/）为您的站点优化图像。这可能需要您更新内容的结构，另请参见[内容组织]（https://gohugo.io/content management/organization/）。使用image和smallimg快捷方式充分利用图像优化。

帖子用标记和LaTeX格式书写（用于数学符号和方程式）。你可以在网上找到大量关于如何用标记和LaTeX格式完善你的文章的信息。

进一步定制

若要更改列表页顶部的标题和子文本，只需在生成列表页的文件夹中添加一个_index.md文件。 例如，要更改主页顶部的标题，请使用以下参数向内容文件夹中添加一个 _index.md文件。

---
title: "可以包含大写字母的标题"
date: 日期
description: "这是主标题上方的潜台词，用小写字母表示"
---
